# Vue2.x 豆瓣Web App

> A project based on Vue.js 

## Build Setup

``` bash

git clone https://github.com/Agressif/Vue2-douban.git

cd Vue2-douban

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

```
